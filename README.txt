
MEESHOP – UI/UX REVAMP PROJECT
================================

Upgraded college project version with:
- 50 real product cards across 10 categories
- Real product/category photos using public image URLs
- Responsive homepage, product listing, product detail, cart, checkout, payment, auth, wishlist, orders, and profile flow
- Dynamic category filters and counts
- Product detail gallery thumbnails
- Added assets folder structure for future local files

HOW TO RUN
----------
Open index.html in any browser.
No backend/server required.
Internet connection is recommended because product images use online photo URLs.
